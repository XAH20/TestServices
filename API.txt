Сервисы работают по протоколу WebSocket

Endpoints:
Соединение /ws
Запросы /app
Ответы /topic

1. Сервис получения названия месяца по его номеру

MessageMapping - /app/getMonth
SendTo - /topic/month
Параметр "month" - число в интервале от 1 до 12

Пример 1:

Request: 8

Response:
А-В-Г-У-С-Т

Пример 2:

Request: 0

Response:
INCORRECT INPUT DATA

2. Сервис преобразования и сортировки строк

MessageMapping - app/sortStrings
SendTo - /topic/strings

параметр "strings" - массив строк в формате JSON в виде строки

Пример:

Request:
            ["Тихо струится река серебристая",

            "В царстве вечернем зеленой весны.",

            "Солнце садится за горы лесистые.",

            "Рог золотой выплывает луны."]
}

Response:
[
    "(27): Рог золотой выплывает луны.",
    "(30): Тихо струится река серебристая",
    "(32): Солнце садится за горы лесистые.",
    "(33): В царстве вечернем зеленой весны."
]
